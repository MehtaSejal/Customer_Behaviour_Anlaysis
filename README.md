# Customer Behaviour Analysis

## Overview

Analysed customer behaviour to uncover trends, segments, and actionable insights that support marketing, retention, and revenue growth. The project demonstrates a complete end-to-end data analytics workflow.

## Tools & Workflow

* **Python:** Data loading, cleaning, and EDA
* **SQL (MySQL):** Querying and analysing structured data
* **Power BI:** Interactive dashboards
* **Gamma:** Stakeholder presentation (PPT)
* **Excel:** Initial data validation

**Steps:** Load → Clean → Explore → Analyse → Dashboard → Report → Presentation

## Key Insights

* Identified high-value and repeat customer segments
* Revealed purchasing trends and engagement patterns
* Analysed promotional ROI for pricing and inventory optimisation

## Business Value

* Supports targeted marketing and retention strategies
* Enables data-driven decisions on pricing, stock, and promotions
* Provides clear dashboards and presentations for stakeholders

## How to Run

1. Clone the repo
2. Install Python libraries: `pip install pandas numpy matplotlib seaborn`
3. Run Jupyter notebooks for EDA and cleaning
4. Load cleaned data into MySQL and run queries
5. Explore dashboards in Power BI
